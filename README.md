# Diablo 2 Resurrected Direct Launcher (D2R Direct Launcher)

![D2R Direct Launcher Icon](https://img.shields.io/badge/D2R-Direct%20Launcher-blue)

A ultra-lightweight, seamless one-click launcher for Diablo 2: Resurrected that bypasses the "Play" button on the Battle.net app while keeping you perfectly online.

디아블로 2: 레저렉션 접속 과정에서 은근히 거슬리는 '배틀넷 런처 플레이 버튼 클릭' 절차를 완벽하게 없애주는 원클릭 다이렉트 실행기입니다. 배틀넷 클라이언트 서버와 연동된 공식 `--exec="launch OSI"` 매개변수를 사용하여, 게임 메모리 변조나 오프라인 모드 접속 없이 가장 안전하고 빠르게 접속을 진행합니다.

# 최근 업데이트 내역 (v1.1)
- **스마트 자동 경로 탐색**: 기존처럼 구체적인 `Battle.net.exe` 파일을 직접 찾을 필요 없이, **배틀넷이 설치된 드라이브나 상위 폴더(예: C드라이브)만 지정해 주면 백그라운드에서 알아서 실행 파일을 찾아냅니다.**
- **휴대성 강화(Portable)**: 저장 방식이 윈도우 레지스트리에 종속되지 않고, 실행 프로그램과 같은 위치에 `config.ini` 파일을 생성합니다. 폴더째로 들고 다니며 스팀덱, 기타 PC 환경에서 포터블하게 사용할 수 있습니다.

# How to Use (사용 방법)

  1. `D2R_DirectLauncher.exe`를 다운로드 받아 적당한 장소에 둡니다.
  2. 다운로드 받은 `exe` 파일을 더블 클릭으로 실행합니다.
  3. (초기 1회에 한해 컴퓨터에 설치된 **드라이브**를 묻는 팝업이 뜹니다. C드라이브 등 배틀넷이 있는 드라이브/폴더를 한 번만 찍어주시면 완료됩니다. 이후 같은 폴더 안에 `config.ini` 파일이 생성됩니다.)


# 스팀덱 (Steam Deck) 활용법

  1. 데스크탑 모드에서 `D2R_DirectLauncher.exe`를 적당한 위치에 다운로드 합니다. (디아블로2 게임과 같은 폴더에 있을 필요 없음)
  2. 다운받은 `D2R_DirectLauncher.exe`파일을 스팀 외부게임 추가를 합니다. ( 마우스 우측 눌러서 스팀숏컷 )
  3. 프로톤 강제 설정에서 현대 디아블로2가 작동되는 프로톤으로 하면됩니다. 모르면 그냥 최신버전같은걸로 해주세요.
  4. 실행하면 처음 1회는 설치된 경로를 잡아줘야 합니다. 배틀넷이 설치되어있는 드라이브/폴더(보통 compatdata 내부 환경)로 연결해주세요.
  5. 정상적으로 했다면 배틀넷 런처가 실행됩니다. 실행 이후 게임 업데이트는 모두 해주세요.
  6. 여기까지 하고 게임 종료해주고 다시 실행하면 바탕화면에서 즉시 다이렉트 실행됩니다.


# 조건 및 팁
  - 프로그램은 배틀넷을 백그라운드로 띄운 후 디아블로2 실행 명령전달까지 약 15초의 딜레이를 주고 있습니다. 이는 배틀넷 런처 로딩 과정을 보장하기 위함입니다.
  - 하지만 스팀덱이나 UMPC 등 기기 성능 및 네트워크 상태에 따라 런처 실행시간이 이보다 더 오래 걸렸다면 다이렉트 실행이 타이밍 문제로 씹힐 수 있습니다.
  - 그럴 때는 당황하지 마시고, 그냥 배틀넷 창에서 [플레이]를 수동으로 눌러주시면 됩니다. (또는 배틀넷을 끄고 다시 실행하면 거의 작동합니다.)
